INTRODUCTION
------------

This module allows you to have a report listing the image styles per image
fields and per view modes on all entities.

The report will have more information if the recommended modules are enabled.

The report can be accessed at admin/reports/image_styles_mapping_report.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/sandbox/florenttorregrosa/2310347
 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2310347


REQUIREMENTS
------------

This module requires the following module:
 * Image (Core)


INSTALLATION
------------

 * Install as you would normally install a contributed drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-8
   for further information.


CONFIGURATION
-------------

The module has no menu or modifiable settings. There is no configuration.
